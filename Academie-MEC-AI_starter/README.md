# Académie MEC AI

Assistant d'apprentissage personnel pour la formation **Manager d'Établissement Commercial**.

## Installation

```bash
npm install
npm run dev
```

Pour lancer en mode application desktop :

```bash
npm run app
```

## Objectifs V0.1

- Structure React/Vite/Electron
- Tableau de bord
- Référentiel de compétences
- Diagnostic
- Cas McDonald's
- Révisions espacées
