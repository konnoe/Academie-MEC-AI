# Design System — Académie MEC AI

## Palette

| Usage | Couleur | Code |
|---|---:|---|
| Primaire / Management | Bleu | `#2563EB` |
| Progression / Succès | Vert | `#16A34A` |
| Attention / À revoir | Orange | `#F59E0B` |
| Erreur / Difficulté | Rouge | `#DC2626` |
| IA / Coaching | Violet | `#7C3AED` |
| Fond sombre | Bleu nuit | `#0F172A` |
| Surface | Gris bleuté | `#1E293B` |

## Règles UI

- Une couleur = une signification.
- Bleu : apprentissage / management.
- Vert : validé / progression.
- Orange : à revoir.
- Rouge : difficulté.
- Violet : IA.
